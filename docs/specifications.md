# Landing Page Specifications

## Purpose
Create a single-page landing page for Adva Hari Perlzug that presents her AI education and consulting services in a clear Hebrew-first, RTL format and drives visitors toward one primary action: making contact.

## Primary Goal
- Generate qualified contact inquiries for lectures, workshops, training, consulting, and AI implementation work.

## Primary CTA
- Contact / leave details for follow-up.

Source wording present in the brief:
- "רוצים להזמין הרצאה לארגון, לעסק או לקהילה?"
- "השאירו פרטים ואחזור אליכם בהקדם."

## Brand / Positioning
- Name: אדוה הרי פרלצוויג
- Main role line: מנגישת טכנולוגיה | מדריכה ויועצת בעולמות ה AI
- Core message: practical, accessible AI guidance for non-technical and mixed-knowledge audiences.
- Teaching approach: less theory, more hands-on practice, live demos, immediate real-world use.

## Non-Negotiable UX / Build Requirements
- Hebrew-first interface text.
- RTL at the document level.
- Fully responsive across mobile and desktop.
- Accessible by default:
  - semantic HTML
  - logical heading order
  - keyboard-accessible interactions
  - visible focus styles
  - sufficient color contrast
  - form labels if a contact form is used
- Static implementation compatible with the current repo structure.

## Service Catalogue
The page should present these service categories as distinct offers.

### 1. Lectures
Offer summary:
- Introductory and practical AI lectures.
- Focus on understanding AI, seeing real examples, and starting to use tools immediately.

Key outcomes mentioned in the brief:
- save time in day-to-day work
- learn leading AI tools such as ChatGPT
- learn prompt writing
- see examples from marketing, service, HR, management, sales, and training
- get actionable ideas for immediate use

Example lecture themes listed in the brief:
- AI למתחילים
- AI בארגון
- בניית סוכני AI וצ'אטבוטים
- AI למנהלים
- AI לשיווק ומכירות

### 2. Hands-On Workshops
Offer summary:
- Practical workshops where participants work on their own computers.
- Can be a one-time workshop or a multi-session training track.

Working principles:
- real-time practice
- applied tasks from participants' own work context
- gradual learning across sessions when relevant
- personal guidance during practice
- emphasis on confidence and independent usage

### 3. AI Training for Organizations
Offer summary:
- Organization-specific AI training that combines inspiration, practice, and implementation.

Format characteristics:
- built around roles, teams, and organizational challenges
- may combine opening lecture, team workshops, and role-specific guidance
- designed to continue beyond the training session itself
- includes follow-up support for implementation and adoption

### 4. Course: AI Agents / Organizational Assistants Without Code
Offer summary:
- Practical course teaching participants to build AI agents and organizational assistants without coding.

Course outcomes from the brief:
- understand AI models and how to get better results
- write professional prompts
- build task-oriented AI agents
- connect agents to documents and knowledge sources
- create automations and integrations
- build chat interfaces
- improve agent quality over time

Target participants named in the brief:
- managers leading innovation
- project and information systems managers
- HR, training, and organizational development staff
- service, operations, and sales staff
- business owners
- anyone wanting to build AI agents without coding

### 5. Business Consulting / AI Implementation
Offer summary:
- Consulting focused on embedding AI into actual business processes.

Scope described in the brief:
- process review
- pain-point identification
- tool selection
- AI agent / chatbot / automation planning or build
- staff enablement
- implementation support

Expected business outcomes:
- time savings
- process improvement
- service improvement
- higher productivity
- freeing time for higher-value work

### 6. Personal Training
Offer summary:
- One-on-one AI guidance for private users, employees, and managers.

Characteristics:
- no fixed syllabus
- based on the participant's real work and goals
- practical and personalized
- suitable for beginners
